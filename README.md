# Advent of Code Solutions

This repository contains my solutions for [Advent of Code](https://adventofcode.com/) challenges.

## Structure

Solutions are organized by year:

```
advent-of-code/
├── 2025/
├── 2026/
└── ...
```

Each year directory contains solutions for that year's puzzles.

## About Advent of Code

Advent of Code is an Advent calendar of small programming puzzles for a variety of skill levels that can be solved in any programming language. Puzzles unlock daily starting December 1st.

Learn more at [adventofcode.com](https://adventofcode.com/).

## Notes

- Each puzzle has two parts
- Solutions should complete in at most 15 seconds on ten-year-old hardware
